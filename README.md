# DIO-EXCEL-XBOX
## Atividades da Santander - Excel com Inteligência Artificial - 2º Semestre na plataforma DIO

Objetivo - O propósito deste projeto é a criação de um dashboard de vendas estratégico utilizando uma base de dados da Xbox. O foco principal está na organização e visualização de dados para responder a perguntas de negócio essenciais, permitindo a transformação de dados brutos em informações relevantes para a tomada de decisão.

Funcionalidades - O painel centraliza indicadores e gráficos de desempenho, como o total de assinaturas para os serviços EA Play Season Pass, Minecraft Season Pass e Xbox Season Pass. Além disso, apresenta uma análise detalhada sobre o impacto financeiro dos cupons distribuídos por categoria. Para facilitar a navegação, o sistema conta com filtros laterais que permitem segmentar todas as informações de forma anual, mensal ou trimestral.

Metodologia - Para garantir a integridade e a fácil manutenção do projeto, a estrutura do arquivo foi dividida em quatro pilares principais: Assets para elementos visuais, Bases para os dados brutos, Cálculos para o processamento e Dashboard para a interface final. O processamento das informações foi realizado através de tabelas dinâmicas, que servem de motor para a geração dos gráficos e métricas apresentadas.
